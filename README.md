# codex-defer

Public Codex marketplace containing the `at` plugin (`/at` and `/defer`).

## Install

1. Add marketplace:

```powershell
codex plugin marketplace add foto-andreas/codex-defer
```

2. Install plugin:

```powershell
codex plugin add at@codex-defer
```

## Update local install

```powershell
codex plugin add at@codex-defer
```
